## Filament Scout Manager: Full Laravel Scout Feature Parity - 2026-03-01

### Release Description

--
This release delivers a major expansion of **Filament Scout Manager** with broad support for Laravel Scout workflows and improved admin experience.
 

#### Highlights

- Added/expanded core Scout management capabilities to cover the full feature set targeted in this milestone.
- Improved indexing workflows with actions for import, flush, refresh, and syncing index settings.
- Enhanced resource coverage for searchable models, synonyms, and query logs.
- Polished UI styling and presentation across package resources and widgets.
- Included reliability fixes from recent updates to reduce configuration and query-log related issues.
   

#### Recommended release tag

`v1.0.0`
